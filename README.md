# RRT-C++

Simulation of RRT (Rapidly-Exploring Random Tree) algorithm written in C++.

<img src="https://github.com/yanlong658/RRT/blob/main/rrt.jpg" width="70%" height="70%">

## Run simulation

```
ros2 run rrt main
```
